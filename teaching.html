Mst Shapna Akter

Teaching

Dr.Shapna's Teaching Area

  • Natural Language Processing
  • Course code: 4180 / 5180

Teaching a course on NLP involves a comprehensive exploration of how machine learning techniques can be applied to enhance NLP defenses. The course typically begins by establishing a solid foundation in machine learning fundamentals, covering both supervised (e.g., classification, regression) and unsupervised (e.g., anomaly detection, clustering) learning methods. Special emphasis is placed on adapting these algorithms to cybersecurity contexts, addressing challenges such as data preprocessing, feature selection, and model optimization tailored to cybersecurity datasets. Practical applications in cybersecurity form a core component of the course. Students delve into critical areas such as anomaly detection, where machine learning algorithms are used to identify unusual patterns or behaviors indicative of potential cyber threats. Another key focus is malware analysis, where techniques like behavioral analysis and code feature extraction are employed to classify and combat malicious software effectively. Intrusion detection systems are explored, equipping students with skills to detect unauthorized access and suspicious activities within network environments using machine learning-driven approaches. Advanced topics expand students' understanding by delving into adversarial machine learning, preparing them to defend against sophisticated attacks aimed at manipulating machine learning models. Deep learning methodologies are also introduced, showcasing their application in tasks like image-based malware detection or analyzing complex network traffic patterns. Discussions on privacy considerations and ethical implications of deploying machine learning in cybersecurity underscore the importance of responsible technology implementation.
The course incorporates hands-on labs and projects to reinforce theoretical concepts with practical experience. Students engage in building and evaluating machine learning models using real-world cybersecurity datasets, simulating scenarios for threat detection and response. Industry insights and case studies provide context, illustrating how leading organizations leverage machine learning to tackle contemporary cybersecurity challenges effectively. Through assessments and practical exams, students demonstrate their ability to apply machine learning techniques critically, ensuring they are well-prepared to contribute to cybersecurity efforts in diverse professional settings.
